# 📊 Greenplum: Лабораторные работы и отчеты

Этот репозиторий содержит PDF-отчеты по всем практическим работам в рамках изучения MPP-СУБД **Greenplum** (на базе PostgreSQL). Здесь собраны материалы от базового DDL до развертывания кластера и оптимизации производительности.

---

## 📚 Содержание лабораторных работ

### Модуль 1: Основы SQL и программирования в Greenplum

#### 🔹 Практическая работа №1.1
**Создание таблиц, загрузка информации в таблицу, анализ объектов**
- Создание таблиц различных типов (HEAP, AO, AOCO)
- Загрузка данных из файлов и внешних источников (COPY, gpfdist)
- Анализ служебной информации (pg_class, gp_toolkit)
- Изучение размера таблиц и распределения данных по сегментам

#### 🔹 Практическая работа №1.2
**Создание ограничений в таблицах, индексов и материализованных представлений**
- Добавление первичных ключей, внешних ключей и CHECK-ограничений
- Создание и сравнение индексов (B-tree, Bitmap, GiST)
- Работа с материализованными представлениями (MATERIALIZED VIEW)
- Обновление данных в материализованных представлениях (REFRESH)

#### 🔹 Практическая работа №1.3
**Создание различных типов функций и хранимых процедур**
- Скалярные функции и функции, возвращающие таблицу (RETURNS TABLE/SETOF)
- Использование PL/pgSQL и SQL-функций
- Хранимые процедуры (CREATE PROCEDURE) с управлением транзакциями
- Функции-агрегаторы и оконные функции внутри UDF

#### 🔹 Практическая работа №1.4
**Создание CTE-запросов, изучение управляющих структур, оконных функций**
- Рекурсивные и нерекурсивные CTE (WITH ... AS)
- Управляющие структуры в PL/pgSQL (IF, LOOP, WHILE, FOR, CASE)
- Оконные функции (ROW_NUMBER, RANK, LAG, LEAD, SUM OVER)
- Оптимизация сложных запросов с использованием оконных функций

### Модуль 2: Администрирование и оптимизация Greenplum

#### 🔹 Практическая работа №2.1
**Развертывание кластера Greenplum**
- Установка Greenplum Database из исходников или пакетов
- Конфигурация узлов (Master, Standby Master, Segment hosts)
- Инициализация кластера (gpinitsystem)
- Запуск и остановка кластера (gpstart, gpstop)
- Проверка состояния кластера (gpstate)

#### 🔹 Практическая работа №2.2
**Создание объектов БД в распределенной среде**
- Создание таблиц с учетом распределения данных (DISTRIBUTED BY / RANDOMLY)
- Выбор ключа распределения для избежания перекосов (skew)
- Создание схем, табличных пространств и ролей пользователей
- Управление расширениями (CREATE EXTENSION)

#### 🔹 Практическая работа №2.3
**Оптимизация доступа к данным при помощи индексации и секционирования**
- Создание партиционированных таблиц (PARTITION BY RANGE/LIST)
- Использование вложенных партиций и подпартиций
- Сравнение сканирования по секциям и полного сканирования таблицы
- Оптимизация запросов с помощью индексов на партиционированных таблицах
- Анализ планов запросов (EXPLAIN ANALYZE)
